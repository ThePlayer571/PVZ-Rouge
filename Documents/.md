## 配置管理方案
### 概要
一共有这些配置数据需要管理：

| 数据名                | 描述                           | 方案             |
| --------------------- | ------------------------------ | ---------------- |
| 卡牌数据              |                                | ScriptableObject |
| 植物数据              | 植物的数据，很散，需要用Global | Global           |
| ArmorData             |                                | ScriptableObject |
| CardLootData(默认)    |                                | Json+Class       |
| ZombieSpawnData(默认) |                                | Json+Class       |
